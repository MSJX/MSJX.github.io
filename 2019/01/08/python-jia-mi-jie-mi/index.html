PyCryptodome包

  • PyCryto应该是Python最著名的第三方库了，奈何PyCryto在2012年的时候就停止更新了。值得庆幸的是，在Python 3.x上有了一个名为PyCryptodome的库来代替PyCryto，下面简单介绍一下PyCryptodome这个库的用法，主要是AES结合RSA使用。

安装

  • Linux

    pip install pycryptodome

  • Windows

    pip install pycryptodomex

    解密过程中的key、nonce、tag必须与cipher_text在加密过程中生成的key、nonce、tag一致，否则会报错。

  • 封装成一个类

    from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

class AESCrypto(object):

    key = get_random_bytes(16)

    # 加密
    def encrypt(self, content):
        data = content.encode('utf-8')
        cipher = AES.new(self.key, AES.MODE_EAX)
        self.nonce = cipher.nonce
        self.cipher_text, self.tag = cipher.encrypt_and_digest(data)
        return self.cipher_text

    # 解密
    def decrypt(self):
        cipher = AES.new(self.key, AES.MODE_EAX, nonce=self.nonce)
        plaintext = (cipher.decrypt(self.cipher_text)).decode('utf-8')
        try:
            cipher.verify(self.tag)
            return plaintext
        except:
            return "key incorrect or message corrupted"

RSA

​ RSA加密算法是一种非对称加密算法>。在公开密钥加密和电子商业中RSA被广泛使用。

​ 该算法基于一个十分简单的数论事实：将两个大素数相乘十分容易，但那时想要对其乘积进行因式分解却极其困难，因此可以将乘积公开作为加密密钥，即公钥，而两个大素数组合成私钥。公钥是可发布的供任何人使用，私钥则为自己所有，供解密之用

  • Linux安装

    pip install rsa

  • 使用

    import rsa

# 加密
def rsa_encrypted(msg):
    # 生成公钥、私钥
    pub_key, priv_key = rsa.newkeys(1024)
    # 明文编码格式
    content = msg.encode('utf-8')
    # 公钥加密
    crypto = rsa.encrypt(content, pub_key)
    return crypto, priv_key

# 解密
def rsa_decrypted(msg, priv_key):
    # 私钥解密
    content = rsa.decrypt(msg, priv_key)
    data = content.decode('utf-8')
    return data
