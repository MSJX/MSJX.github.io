PyCryptodome包

  • PyCryto应该是Python最著名的第三方库了，奈何PyCryto在2012年的时候就停止更新了。值得庆幸的是，在Python 3.x上有了一个名为PyCryptodome的库来代替PyCryto，下面简单介绍一下PyCryptodome这个库的用法，主要是AES结合RSA使用。

安装

  • Linux

    pip install pycryptodome

  • Windows

    pip install pycryptodomex

  • 解密

    from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

key = get_random_bytes(16)

def decrypt(cipher_text):
    cipher = AES.new(key, AES.MODE_EAX, nonce=nonce)
    plaintext = (cipher.decrypt(cipher_text)).decode('utf-8')
    try:
        cipher.verify(tag)
        return plaintext
    except:
        return "key incorrect or message corrupted"

    解密过程中的key、nonce、tag必须与cipher_text在加密过程中生成的key、nonce、tag一致，否则会报错。

  • 封装成一个类

    `python
    from Crypto.Cipher import AES
    from Crypto.Random import get_random_bytes

    class AESCrypto(object):

    key = get_random_bytes(16)

  # 加密
  def encrypt(self, content):
      data = content.encode('utf-8')
      cipher = AES.new(self.key, AES.MODE_EAX)
      self.nonce = cipher.nonce
      self.cipher_text, self.tag = cipher.encrypt_and_digest(data)
      return self.cipher_text

  # 解密
  def decrypt(self):
      cipher = AES.new(self.key, AES.MODE_EAX, nonce=self.nonce)
      plaintext = (cipher.decrypt(self.cipher_text)).decode('utf-8')
      try:
          cipher.verify(self.tag)
          return plaintext
      except:
          return "key incorrect or message corrupted"
